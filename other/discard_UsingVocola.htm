Direct Loading and Discarding

Sometimes it's useful to reload or discard your command files explicitly. For example, if you modify an include file, Vocola doesn't know enough to reload all command files that use that include file. Likewise, if you delete or rename a command file, the old commands from that file will remain until explicitly discarded or Vocola restarts. You can do loading and discarding yourself using the following commands:

When you say Vocola will
"Load All [Voice] Commands" Reload all commands you have defined.
"Load [Voice] Commands" Reload commands for the currently running application.
"Load Global [Voice] Commands" Reload global commands.
"Discard Old [Voice] Commands" Discard all old commands by discarding all loaded commands then reloading all commands you have defined.
